# My Config Files

## Git
  * Add `.gitconfig` file to `~`
  * Set global email: `git config --global user.email ""`

## MySQL
  * Add `.my.cnf` file to `~`

## Bash
  * Add files to `~`
  * `.bashrc` requires `.aliases` in the same directory
  * `.bash_profile` is required in Bash used by'Git for Windows'
